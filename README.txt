Venoom static site

No npm is needed.

Run locally:
1. Open index.html in your browser.

Or run a local server:
1. cd venoom-static-production
2. python -m http.server 5173
3. Open http://localhost:5173

Files:
- index.html
- features.html
- download.html
- css/styles.css
- assets/*.webp

Change download links:
- Open download.html
- Replace #windows-download, #android-download, and #open-web-app with your real links.
